## 4. 機能要件

<!-- 「〜できること」の形式で記述する。Must/Should/Couldで優先度を分ける -->

- [ ] ユーザーはメールアドレスとパスワードでログインできる
- [ ] ログイン失敗時はエラーメッセージを表示する
- [ ] ログアウトできる

### Should（できれば）
- [ ] パスワードリセット機能を提供する
